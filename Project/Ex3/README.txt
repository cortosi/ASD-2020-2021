****	COMPILAZIONE	***

—— PER COMPILARE LE CLASSI DELL’ESERCIZIO 3 ———

1) posizionarsi in …/Ex3

2) Usando il builder java “Ant” digitate: $ ant all
	In questo modo si andranno a compilare tutti i sorgenti e verrà creato
	l’unico Jar richiesto (quello per l’esecuzione di Junit)

***	ESECUZIONE	***

---PER ESEGUIRE JUNIT PER L’ESERCIZIO 3 —

1) posizionarsi in …/Ex3
2) Usando sempre Ant, digitare: $ ant run_junit
	In queso modo verrà eseguito il test Junit sul package Union_fin
